### 重大变更

<Warning>
此版本包含针对 Beta Webhook 功能用户的次要破坏性变更。
</Warning>

**Webhook 签名验证**（增强安全性所需）：

```javascript
// 之前 (v2.4.x)
app.post('/webhook', (req, res) => {
  const data = req.body;
  // 不验证直接处理
});

// 之后 (v2.5.0)
const crypto = require('crypto');

app.post('/webhook', (req, res) => {
  const signature = req.headers['x-arisk-signature'];
  const payload = JSON.stringify(req.body);
  
  const expectedSignature = crypto
    .createHmac('sha256', WEBHOOK_SECRET)
    .update(payload)
    .digest('hex');
  
  if (signature !== expectedSignature) {
    return res.status(401).send('签名无效');
  }
  
  // 处理已验证的 webhook
});
```

<Steps>
  <Step title="获取 Webhook 密钥">
    导航到 **控制台 > Webhooks** 并复制您的 webhook 密钥
  </Step>
  
  <Step title="更新代码">
    向 webhook 端点添加签名验证
  </Step>
  
  <Step title="测试">
    使用控制台中的"测试 Webhook"按钮进行验证
  </Step>
  
  <Step title="部署">
    部署更新后的代码 - 旧 webhook 将继续工作至 2025 年 1 月
  </Step>
</Steps>
